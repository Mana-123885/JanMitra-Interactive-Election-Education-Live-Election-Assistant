# JanMitra — जनमित्र
### Mera Chunav, Meri Awaaz / मेरा चुनाव, मेरी आवाज़
**"Learn Elections. Play. Vote Smarter."**

JanMitra is a gamified educational application designed to empower Indian citizens with accurate knowledge about the democratic process.

## Features
- **Civic Hero Journey**: Level up from a "Nagarik Seedha" to a "Democracy Champion" by learning and playing.
- **Interactive Process Map**: Explore the 12 critical stages of an Indian election, from registration to government formation.
- **Civic Quiz**: Test your knowledge across multiple categories and earn XP.
- **AI-Powered Assistant**: Ask questions about voting rules, EVMs, and the Constitution.
- **Polling Simulator**: Experience polling day step-by-step before you head to the booth.
- **Myth vs Fact**: Debunk common misconceptions with verified information.
- **Live Tracker**: Stay updated with historical and upcoming election schedules.

## Critical Accuracy
All information in this app is verified against authoritative sources:
- Election Commission of India (ECI)
- Constitution of India
- Representation of the People Act, 1950 and 1951

## Technology Stack
- **Language**: Python 3.x
- **GUI Framework**: CustomTkinter (Premium, Modern UI)
- **Database**: SQLite3 (Local Persistence)
- **Image Generation**: Programmatic UI drawing (No external assets)

## Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```

## Repository Constraints
- Total size remains well under 10 MB.
- Optimized for lightweight performance.
- Clean, modular OOP structure.
